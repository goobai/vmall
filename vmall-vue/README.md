# vue-mall

#说明
本项目为本人学习python过程中作为练手做的一个的商城前端界面，主要用到的技术有：vue+vue-router+vant+flex.

#功能列表
1. 首页
2. 商品搜索
3. 商品展示
4. 商品评价
5. 购物车
